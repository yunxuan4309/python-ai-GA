# Python AI Genetic Algorithms (GA)

一个使用Python实现的遗传算法及相关优化算法的项目，包含符号回归、函数优化等多种应用场景。

## 📋 项目简介

本项目展示了遗传算法在不同问题上的应用，包括：
- 符号回归（Symbolic Regression）
- Rastrigin函数优化
- OneMax问题求解
- 多种优化算法对比实验

## 🚀 功能特性

### 1. 符号回归 (`GA.py`)
使用DEAP库实现符号回归，寻找能够拟合给定数据点的数学表达式。

**主要特点：**
- 基于DEAP框架的遗传编程
- 支持基本算术运算和三角函数
- 自动发现数学表达式结构

### 2. 遗传算法优化 (`genetic_algorithm.py`)
使用遗传算法求解Rastrigin函数的最小值问题。

**主要特点：**
- 二进制编码/解码机制
- 轮盘赌选择算子
- 单点交叉操作
- 位变异操作
- 收敛曲线可视化

### 3. OneMax问题 (`oneMax.py`)
经典的OneMax问题求解示例，用于演示遗传算法的基本原理。

**主要特点：**
- 简单的二进制串优化问题
- DEAP框架基础用法展示
- 双点交叉和位变异

### 4. 算法对比实验 (`comparison_methods.py`)
对比遗传算法与其他优化方法（穷举法、随机搜索）的性能。

**主要特点：**
- 多算法性能对比
- 运行时间和精度分析
- 综合评估报告生成

## 🔬 算法说明

### 遗传算法流程
1. **初始化**: 随机生成初始种群
2. **评估**: 计算每个个体的适应度值
3. **选择**: 根据适应度选择优秀个体
4. **交叉**: 对选中个体进行基因重组
5. **变异**: 以一定概率改变个体基因
6. **迭代**: 重复步骤2-5直到满足终止条件

### 参数配置
- 种群大小: 20-300
- 进化代数: 10-200代
- 交叉概率: 0.5-0.7
- 变异概率: 0.02-0.2
